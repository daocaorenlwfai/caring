# Caring 微信小程序 - UI设计规范

## 1. 设计概述

- **产品名称**：Caring 亲友关怀提醒
- **产品类型**：微信小程序
- **目标用户**：25-40岁女性
- **设计风格**：温暖、亲切、简洁

---

## 2. 色彩规范

### 主色调

| 颜色名称 | 色值 | 用途 |
|---------|------|------|
| 主色（珊瑚粉） | `#FF7F7F` | 按钮、图标、高亮 |
| 浅粉 | `#FFF0F0` | 背景、卡片 |
| 深粉 | `#E85A5A` | 按钮按下状态 |

### 辅助色

| 颜色名称 | 色值 | 用途 |
|---------|------|------|
| 温暖米 | `#FFF8F0` | 页面背景 |
| 柔和灰 | `#F5F5F5` | 分割线、禁用态 |
| 浅灰 | `#999999` | 次要文字 |
| 深灰 | `#333333` | 主要文字 |
| 纯黑 | `#000000` | 强调文字 |

### 语义色

| 颜色名称 | 色值 | 用途 |
|---------|------|------|
| 成功绿 | `#52C41A` | 成功状态 |
| 警告橙 | `#FA8C16` | 提醒状态 |
| 错误红 | `#FF4D4F` | 错误提示 |

---

## 3. 字体规范

- **主字体**：系统默认字体（微信小程序兼容性好）
- **标题字号**：20px（页面标题）、18px（模块标题）
- **正文字号**：14px
- **辅助字号**：12px
- **字重**：Regular 400 / Medium 500

---

## 4. 间距规范

- **页面边距**：16px
- **模块间距**：12px
- **元素间距**：8px
- **小间距**：4px

---

## 5. 圆角规范

- **大圆角**：16px（卡片、弹窗）
- **中圆角**：12px（按钮、输入框）
- **小圆角**：8px（标签、小元素）

---

│  │         [关心一下]   ││
│  └─────────────────────┘│
│  ┌─────────────────────┐│

│  └─────────────────────┘│

**组件说明**：

- 日期大字：28px，加粗
- 亲友卡片：
  - 白色背景，圆角12px，阴影
  - 左侧头像（圆形，40px）
  - 右侧事件信息和提醒时间
  - 底部"关心一下"按钮（珊瑚粉填充，白色文字）

## 7. 组件状态

### 按钮

| 状态 | 背景色 | 文字色 | 边框 |
|-----|-------|-------|-----|
| 正常 | `#FF7F7F` | `#FFFFFF` | 无 |
| 按下 | `#E85A5A` | `#FFFFFF` | 无 |
| 禁用 | `#F5F5F5` | `#999999` | 无 |

### 输入框

| 状态 | 背景色 | 边框 |
|-----|-------|-----|
| 正常 | `#FFFFFF` | `#E5E5E5` |
| 聚焦 | `#FFFFFF` | `#FF7F7F` |
| 错误 | `#FFFFFF` | `#FF4D4F` |
