# Caring - 亲友关怀提醒工具 设计稿

## 项目概述

**产品名称**：Caring（关心）
**产品类型**：微信小程序
**目标用户**：个人使用，给老婆的专属工具
**核心功能**：记录亲友的重要事件 + 定时提醒关心近况

---

## 配色方案

### 主色调（温暖、温馨）

| 用途 | 颜色名称 | 色值 |
|------|----------|------|
| 主色 | 温暖橙 | #FF9B6A |
| 副色 | 淡雅粉 | #FFB5B5 |
| 强调色 | 活力橙 | #FF7D4D |
| 背景色 | 米白色 | #FFF9F5 |
| 卡片背景 | 纯白色 | #FFFFFF |
| 文字主色 | 深棕色 | #5D4037 |
| 文字次色 | 中棕色 | #8D6E63 |
| 边框色 | 浅粉色 | #FFE4E1 |

### 渐变方案
- 按钮渐变：linear-gradient(135deg, #FF9B6A 0%, #FF7D4D 100%)
- 背景渐变：linear-gradient(180deg, #FFF9F5 0%, #FFF0E8 100%)

---

## 字体方案

- 主字体：-apple-system, BlinkMacSystemFont, "PingFang SC", "Microsoft YaHei", sans-serif
- 标题：28px (页面标题), 20px (卡片标题), 16px (列表标题)
- 正文：14px
- 辅助文字：12px

## 组件规范

### 按钮

| 类型 | 样式 |
|------|------|
| 主按钮 | 背景: 渐变(#FF9B6A→#FF7D4D), 圆角: 24px, 文字: 白色 |
| 次按钮 | 背景: 透明, 边框: 1px #FF9B6A, 文字: #FF9B6A |
| 文字按钮 | 文字: #FF9B6A, 无背景 |

### 卡片

- 背景: 白色
- 圆角: 16px
- 阴影: 0 4px 12px rgba(255,155,106,0.1)
- 内边距: 16px

### 输入框

- 背景: #FFF9F5
- 边框: 1px solid #FFE4E1
- 圆角: 12px
- 聚焦边框: 1px solid #FF9B6A
- 内边距: 12px 16px

### 标签

- 背景: #FFF0E8
- 文字: #FF7D4D
- 圆角: 12px
- 内边距: 4px 12px

## 交互动效

1. **页面切换**：左滑/右滑，300ms ease-out
2. **按钮点击**：缩放0.95，100ms
3. **卡片出现**：淡入 + 上移，200ms
4. **弹窗显示**：缩放 + 淡入，200ms
